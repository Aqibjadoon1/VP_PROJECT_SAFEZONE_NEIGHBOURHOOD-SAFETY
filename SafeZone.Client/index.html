Why SafeZone?

Built for CS-284L Visual Programming Lab with cutting-edge web technologies

🗺️

Live Map Integration

Interactive Leaflet.js map with real-time incident markers, heatmaps, and proximity-based alerting.

🆘

AI Emergency Calling

One-tap SOS connects you to police, ambulance, or fire brigade. GPT-4o generates the emergency script automatically.

📄

Digital FIR System

File First Information Reports digitally with our 4-step wizard. QuestPDF generates official PDFs for your records.

Real-Time Updates

SignalR-powered WebSocket connections for instant incident notifications, status updates, and proximity alerts.

Authority Dashboard

Kanban-style incident management, heatmap analytics, and broadcast alerts for police and rescue personnel.

Secure & Anonymous

JWT authentication with optional anonymous reporting. Your identity is protected when you need it.
